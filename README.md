# KevinBacon
requires Java 11, Gradle 5
Build Instructions:
./gradlew clean build

Run Instructions:
java -jar ./build/libs/KevinBacon-1.0-SNAPSHOT.jar

example input parameters
java -jar ./build/libs/KevinBacon-1.0-SNAPSHOT.jar 'Footloose (1984 filim)' 'Kevin Bacon'
